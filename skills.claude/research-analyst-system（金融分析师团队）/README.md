# 宏观投资研究Agent系统 使用指南

## 系统概述

这是一个三层级协作的AI分析师Agent系统，专门用于2026年宏观经济与投资策略研究。

```
用户
  ↓
首席分析师 Agent (用户界面 + 任务协调)
  ↓
┌─────────┬─────────┬─────────┬─────────┬─────────┬─────────┐
│内资股票 │外资股票 │大宗商品 │加密货币 │中国宏观│美国宏观│
│小组长  │小组长  │小组长  │小组长  │小组长  │小组长  │
└─────────┴─────────┴─────────┴─────────┴─────────┴─────────┘
  ↓        ↓        ↓        ↓        ↓        ↓
研究员N+ 研究员N+ 研究员N+ 研究员N+ 研究员N+ 研究员N+
```

## 快速开始

### 使用方法

你有3种方式启动研究系统：

#### 方式1: 使用 Slash Command（推荐）
```
/analyze 黄金投资机会
```
或
```
/research AI芯片分析
```

#### 方式2: 使用 Skill
```
/research-analyst-system
```
然后在对话中提出你的研究问题

#### 方式3: 直接对话
直接向浮浮酱提出研究问题，系统会自动识别并调用

### 提问示例

**单一品种分析**:
```
请分析2026年黄金的投资机会，包括具体标的和驱动逻辑
```

**跨领域对比**:
```
对比分析紫金矿业和巴里克黄金
```

**综合组合**:
```
我想要一个"三硬资产"的懒人投资组合
```

**宏观影响**:
```
美联储降息对A股有什么影响？
```

### 研究过程

1. **首席分析师**理解你的问题并拆解任务
2. **并行调用**相关研究小组（可能同时调用多个）
3. 各**小组长**分配任务给研究员
4. 研究员**并行搜集和分析**资料
5. 小组长**审查汇总**研究员报告
6. 首席分析师**整合输出**最终报告

### 报告输出

研究报告自动保存到 `/report/YYYYMMDD_HHMMSS_主题/` 目录，包含：
- `00_首席分析师报告.md` - 综合结论和投资建议
- `01-06_各小组/` - 各小组的详细研究
  - `小组长报告.md` - 小组综合分析
  - `研究员01_任务名称.md` - 各研究员独立报告

### 示例1：单品种分析
```
用户: 请分析2026年黄金的投资机会

系统执行:
1. 首席分析师拆解任务 → 大宗商品小组

   - 研究员04: 具体投资标的
3. 汇总生成报告到 report/YYYYMMDD_HHMMSS_黄金投资分析/
```

### 示例2：跨领域综合研究
```
用户: 我要一个"三硬资产"投资组合

系统执行:
1. 首席分析师拆解任务 → 全部6个小组
2. 并行调用所有小组长
3. 每个小组长分配2-5名研究员
4. 首席分析师汇总所有小组报告
```

```
用户: 对比分析紫金矿业和巴里克黄金

系统执行:

```

- **历史教训**: 冷战结束后，战争金属价格回落20年
- **转折信号**: 关注地缘缓和信号

### 桥水达利欧 (Ray Dalio)
- **2025年总结**: 所有法币都在贬值，黄金表现最佳
- **核心判断**: 美元相对黄金暴跌39%
- **关注重点**: 债务周期、货币体系变革、AI生产力

## 输出质量保证

### 三级审查机制

**研究员级别**:
- 必须引用具体数据来源
- 区分事实与观点
- 标注不确定性

**小组长级别**:
- 交叉验证多个研究员结论
- 剔除矛盾信息
- 确保逻辑一致性

**首席分析师级别**:
- 检查各小组结论一致性
- 识别跨领域机会
- 平衡不同观点

### 数据验证原则

- 优先使用官方数据源
- 多源交叉验证
- 标注数据时效性
- 区分预测与事实

## 注意事项

1. **报告路径**: 所有报告自动保存到 `/report/` 目录
2. **时间戳**: 每次研究生成新目录，便于历史对比
3. **版本控制**: 同一主题更新时创建新目录，旧报告保留
4. **风险提示**: 所有投资建议都包含风险因素
5. **非投资建议**: 报告仅供参考，不构成投资建议

## 常见问题

**Q: 为什么我的问题只触发了部分小组？**
A: 首席分析师会根据问题内容智能判断需要调用哪些小组。只调用相关小组可以提高效率。

**Q: 报告保存在哪里？**
A: 所有报告保存在项目根目录的 `/report/` 文件夹中，按时间戳和主题组织。

**Q: 我可以自定义研究范围吗？**
A: 可以！在提问时明确指定范围，例如"只从内资股票角度分析XX"。

**Q: 系统会自动搜索最新数据吗？**
A: 是的，每个层级的Agent都可以调用搜索功能获取最新信息。

**Q: 报告可以修改吗？**
A: 可以，所有报告都是Markdown格式，您可以自由编辑和补充。
