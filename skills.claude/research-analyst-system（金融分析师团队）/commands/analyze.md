# analyze

启动宏观投资研究Agent系统，由首席分析师协调6大研究小组进行深度分析。

## 使用方法

```
/analyze [你的研究问题]
```

## 示例

```
/analyze 黄金2026年投资机会
/analyze 对比紫金矿业和巴里克黄金
/analyze 三硬资产投资组合配置
```

## 系统架构

```
用户 → 首席分析师 → 6大研究小组（内资股票/外资股票/大宗商品/加密货币/中国宏观/美国宏观）→ 研究员团队
```

## 输出

研究报告自动保存到 `/report/YYYYMMDD_HHMMSS_主题/` 目录，包含：
- `00_首席分析师报告.md` - 综合结论
- `01-06_各小组/` - 分组详细研究
