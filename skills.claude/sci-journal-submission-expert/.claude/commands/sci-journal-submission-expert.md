# /sci-journal-submission-expert

在以下场景使用该命令（或让助手加载同目录 skill）：

- 初稿/修回前的 **预审** 与拒稿风险分级
- **选刊** 与多刊梯度规划、掠夺性期刊风险提示
- 对照 **Author Guidelines** 的格式与声明类核查清单
- **审稿意见** 拆解与 Point-by-Point 回复框架
- **预印本、署名、利益冲突、数据可用性** 等合规边界梳理
- **APC、豁免、发票与报销凭证** 的流程性指引

## 入口

## 不可妥协规则

- 不协助学术不端；不捏造期刊条款或伪造指标。
- 法律与财务段落须带 skill 中规定的免责声明。

调用前请尽量说明：**投稿阶段**、**稿件类型**、**目标期刊（如有）**、**当前最紧迫问题**。
