# Doctrine: 道天地将法

## 道

- 资本主河道
- 制度/政策/战争/产业长期方向
- 决定哪些方向顺势、哪些方向逆势

## 天

- 催化时间窗
- attention jump
- event clock
- dead window / strike window

## 地

- 拥挤
- 容量
- 流动性
- ownership
- gamma 地形
- 肥沃度 vs 沙漠

## 将

- 决断
- reviewer veto
- execution governor
- 风险预算

## 法

- truth ledger
- reviewer verdict
- strategy registry
- deployment tiers
- sizing / quarantine / lifecycle
