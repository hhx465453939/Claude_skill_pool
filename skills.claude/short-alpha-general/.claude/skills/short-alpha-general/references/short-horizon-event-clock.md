# Short-Horizon Event Clock

短周期 alpha 的核心不是“知道什么”，而是“知道市场什么时候会被迫知道”。

事件时钟必须覆盖：

- earnings / guidance
- OPEX / expiry
- rebalance / month-end / quarter-end
- policy / tariff / sanctions
- product launch / ship date / keynote
- litigation / ruling / vote
- dealer gamma / positioning windows

输出必须区分：

- strike window
- probe window
- dead window
