# Submission Checklist

Use this checklist to verify readiness before submission.

## Manuscript
- Title, author list, affiliations, and corresponding author details are consistent.
- Abstract and keywords match the target journal requirements.
- Figure and table callouts in text match file names.
- References are complete and consistent with the journal style.

## Figures and Tables
- Files are in the expected format and naming pattern (e.g., `Fig.1.pdf`).
- Resolution and labeling meet journal guidance.
- Legends are complete and consistent with the manuscript text.

## Submission Forms
- Conflict of interest and funding statements are filled.
- Suggested reviewers, classifications, and questionnaires are completed.
- Any required declarations are signed or finalized.

## Package Integrity
- All required files are present in the target submission folder.
- Dates, journal name, and submission type are consistent across documents.
